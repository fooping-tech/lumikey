光る1キー・キーボード

Lumikey Assembly Guide

Lumikeyは光と遊ぶ1キー・キーボード型キーホルダー。ここでは、初めての方でも安心して組み立てられるように手順とヒントをまとめました。

コンセプト

Lumikeyは、スイッチを光らせるための最小構成にこだわったワンキー・キーボードです。キーキャップを好みの色で発光させたり、ショートカットを割り当てたり、ちょっとしたはんだ付けの練習にもぴったり。組み立ては手のひらサイズで完結します。

部品リスト

必須部品

  • Lumikey PCB
  • メカニカルキースイッチ (MX互換)
  • キーキャップ (1Uサイズ推奨)
  • LED (アドレス指定型、例: SK6812mini-E)
  • ダイオード (1N4148 など)
  • USB Type-C ケーブル

あると便利

  • フラックス付きはんだ
  • ピンセットまたはスイッチプラー
  • キーキャッププラー
  • シリコンマットや耐熱シート
  • テスター (導通チェック用)

組み立て前チェック

準備ポイント

作業スペースを確保し、静電気対策として金属部分に触れておきましょう。LEDの向き (データIN/OUT) とダイオードの方向 (ライン側) を確認し、必要な工具が揃っているかチェックします。

組み立て手順

  1. PCBを取り出し、部品面と裏面のシルクを確認します。LEDとダイオードの向きがシルクに示されているかチェックします。
  2. LEDを指定のパッドに仮置きし、はんだ付けします。熱をかけすぎないよう10秒以内で作業しましょう。
  3. ダイオードをシルクのラインに合わせて配置し、足を曲げて固定したらはんだ付けします。
  4. 必要に応じてマイコンやリセットボタンなど、他の表面実装部品を先にはんだ付けします。
  5. スイッチピンにフラックスを塗布し、PCBにはめ込んだ状態で各ピンをはんだ付けします。
  6. USBケーブルを接続し、LEDが点灯するか、スイッチが入力を検出するかをテスターやPCで確認します。
  7. キーキャップを装着し、ぐらつきがないかを確かめます。必要なら潤滑油でスイッチを調整してください。
  8. 完成品を清掃し、ケーブルに取り付けて光の演出を楽しみましょう。

トラブルシューティング

LEDが光らない場合

  • LEDの向きと極性を再確認します。
  • データラインのはんだ付けが切れていないかチェックします。
  • マイコンのファームウェアが正常に書き込まれているか確認します。

キー入力が反応しない場合

  • スイッチのピンがしっかりはんだ付けされているか確認します。
  • ダイオードの向きが逆になっていないか見直します。
  • USBケーブルや接続機器を別のポートに変えて試します。

カスタマイズのヒント

  • LEDカラーを分割制御できるファームウェアを使用すれば、光り方の演出を自由に設定できます。
  • キーキャップの材質や印字を変えて、持ち歩きたくなるデザインに仕上げましょう。
  • アクリルプレートや3Dプリントケースで保護すれば耐久性アップに繋がります。
